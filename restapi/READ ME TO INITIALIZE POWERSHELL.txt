If your PowerShell script refuses to connect to the remote server, try these two tips:

Run PowerShell as Administrator. Enter "Set-ExecutionPolicy RemoteSigned". This get around one barrier. Exit.

Open the PowerShell windows where you will be working. Before running any scripts, enter "[Net.ServicePointManager]::SecurityProtocol = [Net.SecurityProtocolType]::Tls -bor [Net.SecurityProtocolType]::Tls11 -bor [Net.SecurityProtocolType]::Tls12".  The gets around TLS version errors. 

It is not clear to me how often these incantations must be repeated. -- Bruce Clayton
